Aufgabe: Kinoticket-Buchungssystem

Beschreibung:
Ein Kino möchte ein Online-System entwickeln, über das Besucher Kinotickets bequem von zu Hause aus kaufen können.

Unsere Aufgabe:

Formuliere 1 User Story für dieses System.
Verwende dabei die typische Struktur:

Als [Rolle] möchte ich [Ziel/Wunsch], damit/um [Nutzen/Zweck]

Gebe auch Akzeptanzkriterien an

  User Story: Kinoticket-Buchungssystem
#
# Als Kinobesucher
# möchte ich online ein Ticket für einen gewünschten Film, Sitzplatz und Filvorstellung buchen können,
# damit ich mir meinen Platz im Kino im Voraus sichern und Wartezeiten an der Kasse vermeiden kann.
#
#  Akzeptanzkriterien:
# 1. Der Benutzer kann eine Filmvorstellung (Film, Datum, Uhrzeit) aus einer Liste auswählen.
# 2. Das System zeigt verfügbare Sitzplätze im Saal an.
# 3. Der Benutzer kann einen oder mehrere Sitzplätze auswählen.
# 4. Der Benutzer kann die Buchung bezahlen (z. B. per Kreditkarte oder PayPal).
# 5. Nach erfolgreicher Zahlung erhält der Benutzer eine Buchungsbestätigung 
#    mit QR-Code oder Ticketnummer per E-Mail.
# 6. Das System verhindert, dass derselbe Sitz doppelt gebucht wird.
